El proyecto ConnecTdea se enfoca en el desarrollo de una plataforma web llamada Forumweb software que facilita la comunicación y colaboración académica entre estudiantes y profesores. La aplicación está diseñada con React en el front-end y Firebase en el back-end, tecnologías seleccionadas por su capacidad para crear interfaces de usuario intuitivas y sistemas robustos de gestión de datos. El sistema se enfoca en mejorar la gestión de discusiones académicas, permitiendo la creación, seguimiento y moderación del foro de manera eficiente, con un enfoque la accesibilidad.
